Энэхүү хэсэгт бид нар яаж өөрсдийнхөө веб хуудсыг яаж хүмүүст аятайхан шиг харагдуулах вэ гэдэг зүйлийг CSS ашиглан хийх юм. CSS нь веб хуудсанд тухайн агуулга яаж харагдах вэ гэдэг дүрмийг зааж өгдөг. Жишээ нь: тухайн веб хуудасны ар талын өнгө нь гялгар өнгөтэй, бүх агуулга нь саарал өнгөтэй тэгээд бас Arial хэлбэрийн фонттой эсвэл зарим нэг гарчиг нь хөх өнгөтэй, italic, Times гэдэг фонттой байх гэх мэт. Хэрвээ та нэг удаа CSS дүрмийг сурчихвал, CSS нь үнэндээ түүний дүрмийн тохиргоонуудыг л янз бүрээр хооронд нь уялдуулан ашигласнаар веб хуудсаа хийж болох юм гэдгийг ойлгоно.
